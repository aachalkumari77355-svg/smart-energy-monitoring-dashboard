About the Project

The Smart Energy Monitoring Dashboard is a web-based application designed to track electricity consumption, calculate energy costs, and visualize daily usage trends. It helps households and offices reduce power wastage and improve energy efficiency.

Key Features

  • Appliance-wise energy calculation
  • Total electricity consumed per day
  • Energy cost estimation
  • Previous 6 days usage tracking
  • Interactive charts and tables
  • CSV export of usage data

Technologies Used

Frontend: HTML, CSS, JavaScript, Chart.js
Backend (Future Scope): Node.js / Flask for database integration
